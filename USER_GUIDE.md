## ğŸ“¡ Offline Mode

### **How It Works:**
1. **First Visit (Online):**
   - App downloads hospital data
   - Saves to device storage (LocalStorage)
   - Size: ~140KB

2. **Offline Use:**
   - All features work without internet
   - Search, filter, view details
   - Call and directions still work
   - Data from last sync

   - No action needed from user

### **Problem: Location not working**

- Check phone settings â†’ App permissions
- Enable location for browser

### **Problem: Data not loading**

- Clear browser cache

- Check phone has service
- Verify number is correct
- Try alternative emergency numbers

- Install Google Maps app
- Enable browser location permission
- Check internet connection
- Try clicking directions again

### **Problem: App slow or freezing**
**Solution:**
- Close other browser tabs
- Clear browser cache
- Restart browser
- Update browser to latest version

### **Problem: Search not working**
**Solution:**
- Type more than 2 characters
- Check spelling
- Try different search terms
- Clear filters first
